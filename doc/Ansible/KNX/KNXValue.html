Ansible::KNX::KNXValue

a KNXValue is a device-dependant datapoint. It is initialized by a DPT type name (e.g. “1.001“ for binary switch) and is extended by the initializer with the corresponding DPT module (e.g. KNX::DPT1) so as to handle DPT1 frames. Each KNXValue is linked to zero or more group addresses, the first of which will be the “update” value

Attributes

set flag: knxvalue.flags[:r] = true get flag: knxvalue.flags[:r] (evaluates to true, meaning the read flag is set)
